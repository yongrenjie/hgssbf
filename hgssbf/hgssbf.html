Steps for use:

  1. Select your Pokemon.
  2. Select your Pokemon's ability if there is more than one.
  3. Select your opponent's Pokemon.
  4. Select your opponent's Pokemon's ability if you know it.
  5. Select field conditions such as screens, weather, and/or Trick Room.
  6. Select stat boosts or drops, if any.
  7. Select status conditions for you and your opponent, if any.

Disclaimers:

  • As the numbers are taken from STT's Factory Search, stats displayed assume 31 IVs in each stat. The IVs of the Pokemon you have are dependent on the round you are in (1-7), and the number of times you have switched Pokemon. A more detailed description of IVs in the Factory can be found here.
  • Secondary effects of moves are not yet accounted for (i.e. if it says Overheat can 2HKO Miltank, that doesn't take into account the -2 SpA effect of using Overheat the first time)
  • Base power, types, and stats do not change upon selection of different options for nature and moves. This is how it will be for the forseeable future. This is primarily intended for Battle Factory use, and the movesets, natures, and stats of Factory Pokemon are predetermined and invariant. If you want to use this calculator for your own bred/EVed/IVed/hacked Pokemon, the damage formula will still work as long as you provide it with all the correct input, to a certain extent, so use it at your own risk.

Footnote: IVs in the Battle Factory

In Wave 1, you get 6 Set 1 Pokemon and they all have IV0.

In Wave 2, if you swapped 7 times, you get 5 Set 2 Pokemon with IV 4 and 1 Set 3 Pokemon with IV8.
In Wave 2, if you swapped 6 or fewer times, you get 6 Set 2 Pokemon with IV 4.

In Wave 3, if you swapped 14 times, you get 4 Set 3 Pokemon with IV 8 and 2 Set 4 Pokemon with IV12.
In Wave 3, if you swapped between 7 and 13 times, you get 5 Set 3 Pokemon with IV 8 and 1 Set 4 Pokemon with IV12.
In Wave 3, if you swapped 6 or fewer times, you get 6 Set 3 Pokemon with IV 8.

In Wave 4, if you swapped 21 times, you get 3 Set 4 Pokemon with IV 12 and 3 Set 5 Pokemon with IV16.
In Wave 4, if you swapped between 14 and 20 times, you get 4 Set 4 Pokemon with IV12 and 2 Set 5 Pokemon with IV16.
In Wave 4, if you swapped between 7 and 13 times, you get 5 Set 4 Pokemon with IV12 and 1 Set 5 Pokemon with IV16.
In Wave 4, if you swapped 6 or fewer times, you get 6 Set 4 Pokemon with IV 12.

In Wave 5, if you swapped 28 times, you will get 2 Set 5 Pokemon with IV16 and 4 Set 5 Pokemon with IV20.
In Wave 5, if you swapped between 21 and 27 times, you will get 3 Set 5 Pokemon with IV16 and 3 Set 5 Pokemon with IV20.
In Wave 5, if you swapped between 14 and 20 times, you will get 4 Set 5 Pokemon with IV16 and 2 Set 5 Pokemon with IV20.
In Wave 5, if you swapped between 7 and 13 times, you will get 5 Set 5 Pokemon with IV16 and 1 Set 5 Pokemon with IV20.
In Wave 5, if you swapped 6 or fewer times, you will get 6 Set 5 Pokemon with IV16.
